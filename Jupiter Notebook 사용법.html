주피터 노트북¶

주피터 노트북은 웹 브라우저 상에서 개발할 수 있는 도구로서 코드를 cell 단위로 묶어서 실행하고 그래프나 표, 이미지나 영상에 관련된 처리하는데 특화되어 있어서 특히 데이터 관련 작업을 할 때 잘 사용하는 툴이다.

교육을 위한 강의 노트로도 아주 훌륭함

주피터 노트북의 특징¶

  1. 코드를 cell 단위로 작성 및 실행
  2. 마크다운을 통한 문서화
  3. 그래프나 표 등을 실시간으로 확인
  4. html , pdf 등 파일로 저장

시각화예제 2 : 테이블¶

  • pandas는 복잡한 데이터를 분석할 때 매우 유용하다.
In [13]:
import pandas as pd
import numpy as np
import matplotlib as mpl

df = pd.DataFrame({
    "strings": ["Adam", "Mike"],
    "ints": [1, 3],
    "floats": [1.123, 1000.23]
})
df.style \
  .format(precision=3, thousands=".", decimal=",") \
  .format_index(str.upper, axis=1) \
  .relabel_index(["row 1", "row 2"], axis=0)
